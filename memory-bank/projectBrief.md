# Project Brief

## @jbcom/strata

**The complete solution for foreground, midground, and background layer 3D gaming in Node.js**

### Vision
Strata provides everything needed to build high-quality 3D games and experiences, from terrain generation to character animation, optimized for performance across mobile, web, and desktop.

### Core Value Proposition
- **All-in-one**: Terrain, water, sky, vegetation, characters, effects in one package
- **Performance**: GPU-accelerated, mobile-optimized, instanced rendering
- **Framework-agnostic core**: Pure TypeScript algorithms in `src/core/`
- **React Three Fiber components**: Ready-to-use components in `src/components/`
- **Production-ready**: Input validation, error handling, comprehensive tests

### Target Audience
- Game developers using React Three Fiber
- 3D visualization developers
- Interactive experience creators
- Mobile game developers (via Capacitor/React Native plugins)

### Technology Stack
- **Runtime**: React Three Fiber, Three.js
- **Physics**: Rapier 3D
- **Audio**: Howler.js
- **State**: Zustand, XState
- **AI**: Yuka
- **ECS**: Miniplex
- **Build**: TypeScript, pnpm, Vitest, Playwright
