    Class SoundManager

    Manages 2D audio playback using Howler.js.

    Example

    const manager = createSoundManager({ masterVolume: 0.8 });

    await manager.load('explosion', { src: '/audio/explosion.mp3' });
    manager.play('explosion');

    manager.setBusVolume('sfx', 0.5);
    manager.dispose();
